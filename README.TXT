Broad Matlab Toolkit
----------------
v 1.0 "Americone Dream"

Features:
-------
lightweight matlab library
support for Broad file formats
computational biology algorithms
specialized plots

Contributors:
----------
Brian Geier, bgeier@broadinstitute.org
Rajiv Narayan, narayan@broadinstitute.org

Usage:
-----
a. To use run:
addpath(genpath('path/to/bmtk'))
b. To remove from path run:
rmpath(genpath('path/to/bmtk'))
c. If new subfolders were created, need to refresh path (remove and add again)

Style Guidelines:
-------------
Filenames should be all lower case
Compiled scripts should be named: XXX_tool.m
Comandline arguments
-res
-grp_gene
-grp_sample
